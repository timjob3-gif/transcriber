## 🖥 GPU — мониторинг и использование

- [ ] **Определение GPU при старте** — Python проверяет `torch.cuda.is_available()` и `torch.cuda.get_device_name(0)`, отправляет событие в UI: название карты, VRAM, режим (`cuda` / `cpu`)
- [ ] **Индикатор в шапке** — строка вида `GPU: RTX 3060 · 12 ГБ` или `CPU (GPU не найден)`; зелёный если CUDA, серый если CPU
- [ ] **Настройка устройства** — в панели настроек: авто / CPU / CUDA / выбрать конкретный GPU (для машин с несколькими картами)
- [ ] **Выбор compute_type** — `float16` (быстро, CUDA), `int8_float16` (быстро + меньше VRAM), `int8` (только CPU); сейчас выбирается автоматически, дать управление пользователю
- [ ] **VRAM-монитор во время транскрипции** — показывать занятую VRAM в реальном времени через `torch.cuda.memory_allocated()` / `nvidia-smi`; предупреждать если < 1 ГБ свободно
- [ ] **Fallback на CPU** — если на GPU не хватает VRAM, автоматически перезапустить с `device=cpu` и уведомить пользователя
- [ ] **Поддержка нескольких GPU** — передавать `device_index=N` в WhisperModel; показывать список доступных карт в настройках
- [ ] **Онбординг: CUDA toolkit** — при детекции NVIDIA-карты без CUDA предлагать установить `torch` с поддержкой CUDA (`pip install torch --index-url https://download.pytorch.org/whl/cu121`)
