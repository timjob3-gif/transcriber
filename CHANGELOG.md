# Changelog — Транскрибатор

## v1.0.5 — 2026-06-08
**Настройки упрощены**
- Секция «Голоса» удалена из панели Настроек
- Диаризация и количество участников настраиваются для каждого файла отдельно в карточке задания

## v1.0.4 — 2026-06-08
**Staging-карточка: настройка перед стартом**
- Каждый загруженный файл теперь ждёт в staging — авто-старт убран
- Галочка «Диаризация» и выбор количества голосов — отдельно для каждого файла
- Кнопка ▷ Начать запускает один файл; кнопка ▷ Все — запускает все ожидающие
- Отмена pending-задания просто убирает карточку (без статуса «Ошибка»)
- Retry возвращает задание в staging с прежними настройками

## v1.0.3 — 2026-06-07
**Системный трей + per-file количество голосов**
- Иконка в системном трее: закрытие окна прячет приложение, не убивает процессы
- Контекстное меню трея: «Открыть» / «Выйти»
- Выбор количества участников (Авто / 2–6) вынесен в каждую задачу

## v1.0.2 — 2026-06-06
**Переименование спикеров + панель истории**
- Двойной клик на бейдже спикера в просмотрщике → переименование (S0 → «Иван»)
- Кнопка 🕓 в шапке открывает боковую панель истории транскрипций
- Клик по записи в истории открывает её в просмотрщике

## v1.0.1 — 2026-06-05
**Диаризация спикеров**
- Speaker diarization: resemblyzer + webrtcvad + AgglomerativeClustering, без интернета
- Цветные бейджи S0/S1/… в просмотрщике и превью карточки
- Индикатор «🎤 Определяю спикеров…» в карточке
- Настройка: чекбокс диаризации + количество участников в ⚙️

## v1.0.0 — 2026-06-04
**Первый публичный релиз**
- Офлайн-транскрипция через faster-whisper (tiny / base / medium / large-v3)
- Drag-and-drop и выбор файла (MP4, MKV, AVI, MOV, MP3, WAV, M4A, FLAC)
- YouTube-ссылки через yt-dlp
- Экспорт JSON + TXT + SRT + VTT
- Просмотрщик с таймкодами и копированием текста
- Настройки: модель, язык, папка вывода
- GPU-чип в шапке (CUDA / CPU автодетект)
- Прогресс, ETA, скорость ×N в карточке задания
- Welcome-экран со скачиванием модели при первом запуске
- NSIS-инсталлер под Windows x64
